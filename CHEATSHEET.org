#+TITLE: Datum Cheatsheet
#+OPTIONS: toc:nil num:nil

* Quick Start

With the recommended setup, getting from zero to a SQL editing buffer takes
two steps:

1. =M-x sql-connect= (or =M-x sql-datum=) — connect to a database
2. =C-c q= — open a scratch SQL buffer, ready to query

From the scratch buffer you have full completion (=TAB=), all the =C-c s= /
=C-c t= keybindings, and =C-c C-r= / =C-c C-b= to send SQL to the server.

** Recommended init file settings

#+begin_src emacs-lisp
(require 'sql-datum)
(autoload 'sql-datum-scratch "sql-datum" nil t)
(global-set-key (kbd "C-c q") #'sql-datum-scratch)
(add-hook 'sql-mode-hook
          (lambda () (setq-local tab-always-indent 'complete)))
#+end_src

* Emacs Keybindings

** Table Operations (=C-c t=)

| Key       | Action                                       |
|-----------+----------------------------------------------|
| =C-c t e= | Export table to file (=C-u= for custom query)   |
| =C-c t i= | Import file into table (=C-u= for batch size)   |
| =C-c t t= | Select top N rows (=C-u N= for count)        |
| =C-c t c= | Count rows (=COUNT_BIG= on MSSQL)            |
| =C-c t s= | Random sample of N rows (=C-u N= for count)  |
| =C-c t d= | Describe table columns                       |
| =C-c t x= | Check if table exists                        |
| =C-c t D= | Drop table (with confirmation)               |

** Session Info (=C-c s=)

| Key       | Action                   |
|-----------+--------------------------|
| =C-c s p= | Show user/server/db/ver                  |
| =C-c s t= | List tables (=C-u= to filter with LIKE)  |
| =C-c s c= | List columns for table                   |
| =C-c s d= | List databases (=C-u= to filter with LIKE) |
| =C-c s s= | List schemas (=C-u= to filter with LIKE) |
| =C-c s r= | Running queries                          |
| =C-c s v= | Server version                           |
| =C-c s u= | Current user                             |

** Other

| Key     | Action                            |
|---------+-----------------------------------|
| =C-c q= | Open scratch SQL buffer (global)  |
| =C-c u= | Switch database                   |

* Running Queries Buffer

Shows session ID, user, status, duration, and SQL text.
Respects =:chars= for column width. Timestamp shows last refresh.

| Key | Action               |
|-----+----------------------|
| =g= | Refresh              |
| =a= | Toggle auto-refresh  |
| =q= | Quit                 |

* Query Input

| Terminator | Meaning                        |
|------------+--------------------------------|
| =;;=       | End of line, send query        |
| =GO=       | Own line, send preceding lines |
| =?=        | ODBC parameter (prompted)      |

* Built-in Commands

** Session

| Command       | Description                  |
|---------------+------------------------------|
| =:help=       | List all commands            |
| =:pwd=        | User, server, database, ver  |
| =:user=       | Current user                 |
| =:version=    | Server version               |
| =:reconnect=  | Force new connection         |
| =:use <db>=   | Switch database (reconnects on PostgreSQL) |
| =:type [val]= | Show/set dialect             |
| =:exit=       | End session                  |

** Display

| Command          | Description                     |
|------------------+---------------------------------|
| =:rows [n]=      | Max rows (0 = all)              |
| =:chars [n]=     | Max chars/column (0 = all)      |
| =:null [str]=    | NULL display (=OFF= = blank)    |
| =:newline [str]= | Newline replacement (=OFF= = keep) |
| =:tab [str]=     | Tab replacement (=OFF= = keep)  |
| =:timeout [n]=   | Timeout secs (0 = none)         |

** Introspection

| Command          | Description              |
|------------------+--------------------------|
| =:databases [pattern]= | List databases (LIKE filter)  |
| =:schemas [pattern]=  | List schemas (LIKE filter)    |
| =:tables [pattern]=   | List tables/views (LIKE filter) |
| =:columns <tbl>= | Columns (=schema.table=) |
| =:running=       | Running queries          |

** Export / Import

| Command                          | Description                       |
|----------------------------------+-----------------------------------|
| =:out <path> [:force]=           | Export next query (.csv/.parquet/.json) |
| =:in <path> <table>=            | Import (error if exists)          |
| =:in <path> <table> :insert=    | Import (append)                   |
| =:in <path> <table> :replace=   | Import (drop & recreate)          |
| =:in ... :batch N=              | Custom batch size (default 1000)  |
| =:csv [path]=                    | Legacy session CSV export         |
| =:script [path]=                 | Run SQL file                      |

* Connection Examples

#+begin_src shell
# PostgreSQL
datum --driver "PostgreSQL Unicode" --server localhost --database mydb --user postgres --pass ENV=PG_PASS

# MSSQL integrated auth
datum --driver "ODBC Driver 18 for SQL Server" --server myserver --database mydb --integrated

# SQLite
datum --driver SQLite3 --database /path/to/db.sqlite

# DSN
datum --dsn MyDSN

# Connection string
datum --conn-string="Driver={PostgreSQL Unicode};Server=localhost;Database=mydb;Uid=postgres;Pwd=secret"
#+end_src

* Config File Location

1. =--config <path>=
2. =$XDG_CONFIG_HOME/datum/config.ini=
3. =~/.config/datum/config.ini=

* Custom Commands

#+begin_src ini
[queries]
top=SELECT * FROM {table} LIMIT {how_many}
find=SELECT * FROM {table} WHERE {field} = ?
#+end_src

={placeholder}= is prompted and string-formatted. =?= is an ODBC parameter (escaped).
